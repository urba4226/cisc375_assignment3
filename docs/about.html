Tools Used:
  • Leaflet: Leaflet is the API that we used to make our map and add markers to the map. It has all of the scroll, zoom, and drag functionality for the map built in which made it really easy to interface with the application.
  • Nominatim: Nominatim is the API that we used to convert between addresses and latitude/longitude coordinates. We used it alongside Leaflet to create the search functionality for the map, as well as adding markers for individual crimes to the map.
  • Bootstrap: Bootstrap is the framework that we used to format everything with. The table was made using bootstrap, including the color coding of rows depending on crime type. It was also used to create the collapsable filters underneath the map used to filter the crimes displayed.
  • Our RESTful API: We developed this API specifically for this project. It performs queries on the crime database to pull data about neighborhoods, crime codes, and incidents for us to display in the table and on the map.

Interesting Facts Discovered:
  • In October, Captitol River had twice as many total crimes than the next leading areas, with 909 crimes reported
  • In the same date range, however, both Capitol River and Payne/Phalen had the same number of violent crimes (14)
  • Well over half of the crimes in our database fall under "Proactive Police Visits" which is not a crime at all, but a prevention measure and a way the police keep in touch with neighborhoods, buisnesses, and individuals
